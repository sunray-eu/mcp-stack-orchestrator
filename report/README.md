# Report Artifacts

This folder contains the evaluation evidence and final recommendations.

## Main Documents

- `MCP_EVAL_REPORT.md`: full evaluation summary and ranking
- `FINAL_PRODUCTION_RECOMMENDATION.md`: final stack recommendation
- `ARCHON_SURREAL_RUNTIME_VALIDATION.md`: explicit SurrealDB MCP and Archon validation runs
- `WEB_UI_AUDIT.md`: web UI surface checks and runtime accessibility
- `QUICKSTART.md`: operational quickstart
- `DEPLOYMENT_STATUS.md`: deployment baseline snapshot

## Data Files

All tabular raw/derived outputs are in `report/data/`.

Notes:
- Paths were sanitized to placeholders where needed.
- These files are historical records from evaluation sessions.
